# library_system
# Library Management System

## Description
A Python program to manage library books.

## How to Run
1. Open terminal
2. Run: python library_system.py

## Features
- Add a book
- Delete a book
- Modify a book
- Display all books

## Files
- library_system.py  : main program
- library_data.csv   : data file
- description.docx   : system description

## Requirements
- Python 3.x
